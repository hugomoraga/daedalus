# Bounded Context: Commercial

**Purpose:** Turning interest into a committed engagement.

## Domains
- **Lead** *(aggregate root)* — a potential engagement; the entry point of the value chain. Lifecycle: created → qualified → converted (or discarded).
- **Proposal** *(aggregate root)* — a formal offer derived from a qualified Lead. Lifecycle: generated → submitted → approved / rejected. Approval is a governed gate.

See the authoritative definitions in the [Domain Model](../../docs/domain-model.md). Conceptual only in Phase 0.
