# governance/amendments/

Constitutional amendment proposals and history.

Changes to the [Constitution](../../memory/constitution.md) follow the **Article VI** process: proposal → review → human ratification → versioning → propagation. Each ratified amendment is recorded here and emits a `ConstitutionAmended` event.

Two guarantees can never be amended away (Article VI): **Article V** (Human Responsibilities) and **Article IV's** prohibition on agents self-modifying policy. These are the load-bearing constraints that keep the system accountable to humans.

*Empty in Phase 0 — the Constitution v1.0.0 is the founding text; amendments accrue here.*
